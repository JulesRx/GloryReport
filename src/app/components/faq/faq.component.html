FAQ

What is GloryReport ?

GloryReport is an open source project utilising the Destiny API to get various information about your PvP carrier since the game released like the list of players you most played with or against, how much time you won or lose against a player and more.

Who made it ?

GloryReport was made by Myjulot, a Destiny and Halo player.

Is there any future plans for GloryReport ?

GloryReport is the first time i really used the Destiny API seriously and i wish it to not be the last. This application will evolve over time and get better and better, with new features to come. Here's a list of what i want to do, in no particular order :

  • proper theming
  • player vs opponent comparison
  • detailed match report
  • more statistics

Is there any known issues ?

This is the known issues :

  • abominable performances when fetching PGCRs (expecially if a player has more than 8k games)
  • some api calls are returning errors when they shouldn't
  • the team composition is not refreshing when switching between players

If you encounter any problems not listed here, please create an issue on the project's issue tab or contact me on Twitter.

Can i contribute ?

Yes, you can ! If you have an idea of a feature you would like to be implemented, please create a pull-request on GitHub, or contact me on Twitter.
